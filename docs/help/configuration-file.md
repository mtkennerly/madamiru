# Configuration
Madamiru stores its configuration in the [application folder](/docs/help/application-folder.md),
in a file named `config.yaml`.

## Schema
[docs/schema/config.yaml](/docs/schema/config.yaml)

## Example
```yaml
---
release:
  check: true
language: en-US
theme: Dark
playback:
  muted: false
  max: 4
```
